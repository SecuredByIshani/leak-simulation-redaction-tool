# Leak Simulation & Redaction Tool

A web-based tool that simulates data leakage scenarios and demonstrates the risks of improper redaction.

## Features

* Detects sensitive information using regex (emails, phone numbers, IDs)
* Securely redacts data by removing underlying content instead of visual masking
* Extracts EXIF metadata from files to identify hidden information
* Demonstrates common data leakage vulnerabilities

## Tech Stack

* HTML
* CSS
* JavaScript
* Regex
* Metadata Analysis

## Why I Built This

I built this project to explore practical cybersecurity risks involving data exposure and to demonstrate secure redaction techniques.
